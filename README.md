# Universe Spy

## Prepare

    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt

## Configure

    cp parts.txt.example parts.txt
    cp config.txt.example config.txt

Edit `parts.txt` and `config.txt`

## Run

    python server.py

Navigate to [localhost:13337](http://localhost:13337)
